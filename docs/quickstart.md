# Quickstart

```bash
deltaforge list
deltaforge init flashindex --lang rust
cd flashindex-rust
deltaforge overview
deltaforge instructions
deltaforge test
```

After a stage passes, run:

```bash
deltaforge next
```

Use `deltaforge hint` when stuck, `deltaforge bench --save` for local performance history, and `deltaforge report --format markdown --output report.md` for a progress report.
